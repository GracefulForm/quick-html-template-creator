You can use this code to put a colored bar on your page. This is often used under a header image and it matches the menu bar. You can add optional text if you wish. 

In this example, I made the top menu purple, added a standard header image with a tint of purple, and then added this bar below the image with the same purple to tie it all together.

![Add a colored bar with html.](html-color-bar.jpg "This can help add color to your template theme.")

```html

<div class="w3-deep-purple w3-panel" style="margin-top: 0px;">
     <p class="w3-center">&nbsp;</p> <!-- You can add optional text here, or just leave this blank space code to keep the panel from shrinking.-->
</div>

```
