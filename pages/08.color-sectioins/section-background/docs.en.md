You can add a background color to a section by using this panel. In the sample picture, I pasted the "3 Photos" code.

![You can put background color on most sections.](section-background-color.jpg "This is a quick way to add more color or style to your page.")

```html
<div class="w3-blue w3-panel w3-padding-24">
     <!-- paste a section below here -->
    
    <!-- paste a section above here -->
</div>
```
